This is a app to organise your todo's / reminders / tasks all in one place. We'll be adding more features to the todo-task app by including third party api's (Evernote, GIT, googledocs etc). Todo-task app's basic features are add, edit, delete, mark as completed Todo's. We will be adding more features as we keep developing.

  • The Todo's are user specific and you can add Tags, Projects and prioritise Todo's.
  • Todo's can be shared to users.
  • Group the tasks related to a project.
  • Create sub-task's to drill down complex tasks.
